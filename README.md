## 🚀 Projetos em Destaque

### [BDD Studio - jessiqa](https://gitlab.com/jessiqa/bdd-studio)
Plataforma full-stack com IA que gera cenários Gherkin a partir de cartões de requisito, usando RAG (Retrieval-Augmented Generation). Stack: FastAPI, Next.js, PostgreSQL, ChromaDB, Docker e Google Gemini 2.5 Flash. Score de qualidade validado em **8,5/10** em benchmark interno.

### Gerador de Arquivos EDI/CNAB/JSON
Ferramenta web para geração de arquivos nos layouts CNAB 240, CNAB 500, Importação CSV e Script API, com suporte a dados de pagador, fornecedor, vínculo bancário e notas fiscais (até 3000 notas por lote). Inclui deploy automatizado via GitLab CI/CD e versionamento automático por tag. Stack: HTML5, CSS3, JavaScript.

### Formatador de Gherkin
Ferramenta web para formatação e padronização de cenários BDD escritos em Gherkin, desenvolvida como solução interna para aumentar a consistência e qualidade da escrita de testes no time. Precursor direto do BDD Studio - jessiqa.
